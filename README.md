# README

### 项目介绍

- 项目来源: B 站大学 Flask2 Demo 项目
- 项目目的: 个人学习 Python Flask2 web 框架

### 使用方式 

1. clone 本项目到本地 PC
2. 使用 Pycharm 打开项目
3. 在 Pycharm 中运行 app.py 即可


### 维护方式

```
按照日学习进度推送代码到 secret-dev 分支，然后 merge 到 main 主分支
```
